设计心理学 1.0

  • 设计

最近在看一本名叫”设计心理学”的书

之前做设计，就是参考一些案例或者别人的demo，然后再添加一些自己的想法就构成了一个设计，美感从来都是自己看着舒服就行，没想过为什么舒服，为什么不舒服，知道看了这本书。
下面是自己的一些点
1、网页设计需要大面积的留白，这样为了强调中心点的内容
2、在网页中，人们经常会忽略最上边以及左右两边的内容，所以重点的东西要放在中间
3、人们看东西的习惯是从左到右 从上到下
4、你需要引导人们去看重要的东西，暗示、突出等
5、做设计，你要做出他们想要的设计才是对的
6、除去美观之外，要有完整的闭环流程

<4.13
你无法从设计上去弥补产品上的缺陷

谁听说过因为一个很酷炫的效果成就 / 拯救了一家公司？
品牌的价值是由品牌本身的战略和产品决定的，而不是单独的表象层设计。
